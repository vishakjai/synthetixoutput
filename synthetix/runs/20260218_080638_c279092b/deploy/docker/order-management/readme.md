# Order Management Service

This service handles order creation, update, and retrieval, interfacing with MongoDB.

## Requirements
- Node.js
- MongoDB

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the service:
   ```bash
   npm start
   ```

3. Run tests:
   ```bash
   npm test
   ```

## Environment Variables

- `PORT`: The port on which the service will run (default: 8080).
- `MONGODB_URI`: MongoDB connection string.
